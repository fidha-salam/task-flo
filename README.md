# TaskFlo - Academic Assignment Management System

A Python-based desktop application prototype designed to help students efficiently track, manage, and submit academic assignments.

## Overview

TaskFlo simplifies academic task management by automating assignment tracking, deadline monitoring, and assignment submission. It replaces error-prone manual spreadsheet tracking with an intuitive, automated system.

## Features

- **Automated Assignment Tracking**: Import and manage assignments from Excel spreadsheets
- **Deadline Management**: Visual deadline indicators and priority sorting
- **Central Submission System**: Allows file uploads and downloads

## Tech Stack

- **Programming Language**: Python
- **Data Processing**: Pandas library
- **Data Storage**: Excel/CSV files
- **Version Control**: Git & GitHub
